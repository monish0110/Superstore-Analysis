# Superstore Sales Analysis

This is a simple data analytics project based on the Superstore dataset. The goal was to clean the raw data, perform exploratory data analysis (EDA), answer business questions, and build an interactive Power BI dashboard to visualize key insights.

## Tools Used

* Python
* Pandas
* Matplotlib
* Power BI
* Jupyter Notebook

## Project Workflow

### 1. Data Cleaning

The raw dataset was cleaned using Pandas.

Tasks performed:

* Converted date columns into datetime format
* Renamed columns using snake_case
* Checked for missing values
* Checked for duplicate records
* Saved the cleaned dataset for further analysis

### 2. Exploratory Data Analysis (EDA)

Some of the business questions explored:

* Which region generates the highest sales?
* Which region generates the highest profit?
* Which category generates the highest sales?
* Which category generates the highest profit?
* Which sub-categories generate losses?
* Which products generate the highest profit?
* How do discounts affect profit?
* What are the sales trends over time?
* What are the profit trends over time?

### 3. Dashboard Development

An interactive Power BI dashboard was created to summarize key business metrics and insights.

Dashboard includes:

* Total Sales
* Total Profit
* Total Orders
* Total Customers
* Sales Trend
* Profit Trend
* Sales by Region
* Profit by Region
* Sales by Category
* Profit by Category

## Key Insights

* The West region generated the highest sales and profit.
* Technology was the best-performing category in terms of both sales and profit.
* Furniture generated strong sales but comparatively lower profit.
* Some sub-categories consistently generated losses.
* Sales and profit showed an overall upward trend over the years.
